# 访问修饰符 public, private, protected, 以及不写的区别

## private 

在同一类内可见

使用对象：变量、方法 

注意：不能修饰类（外部类） 

## default

在同一包内可见，不使用任何修饰符

使用对象：类、接口、变量、方法

## protected

对同一包内的类和所有子类可见

使用对象：变量、方法

注意： 不能修饰类（外部类） 

## public 

对所有类可见

使用对象：类、接口、变量、方法 

| 修饰符    | 当前类 | 同包 | 子包 | 其他包 |
| --------- | ------ | ---- | ---- | ------ |
| private   | ✔      | ✖    | ✖    | ✖      |
| default   | ✔      | ✔    | ✖    | ✖      |
| protected | ✔      | ✔    | ✔    | ✖      |
| public    | ✔      | ✔    | ✔    | ✔      |
